Features

枝分かれする履歴を、木漏れ日のように見渡す。

See your branching history at a glance — calm, clear, complete.

日々のコミット・ブランチ・マージから、SSH 鍵管理・AI コミットメッセージ生成・GitFlow まで。Git に必要な全てを 17 言語 100% 翻訳で提供します。 From everyday commits and branches to SSH key management, AI commit messages and GitFlow — every Git workflow you need, fully translated into 17 languages.

Visual Commit Graph

ビジュアルコミットグラフ

枝分かれする履歴を立体的に描画。著者・期間・パス・正規表現で高速に絞り込み、追跡したい一筋をすぐ見つけられます。 A dimensional rendering of your branching history. Fast filters by author, date range, path or regex bring the line you're chasing to the surface.

GraphFilterRegexWorktree
GPG · SSH · X.509 Signing

署名コミットに完全対応

GPG・SSH・X.509 いずれの署名方式にも対応。鍵を選び直す煩わしさなく、安心して履歴を残せます。 Sign commits with GPG, SSH or X.509 — pick once, then ship audit-ready history.

17 Languages, 100%

17 言語 100% 翻訳

タガログ・サンスクリット・ラテン語を加えた 17 言語が翻訳率 100%。ロケール別に最適なフォントを自動選択します。 All 17 UI languages — including Tagalog, Sanskrit and Latin — are fully translated, with per-locale font tuning.

AI Commit Messages

AI コミットメッセージ生成

OpenAI / Gemini / Anthropic / DeepSeek / Ollama に対応。差分から的確なメッセージを下書きし、書く時間をぐっと短縮します。 Drafts a precise commit message from your diff — OpenAI, Gemini, Anthropic, DeepSeek and Ollama all supported.

OpenAIGeminiAnthropicOllama
SSH Keys & CodeCommit

SSH 鍵管理 / AWS CodeCommit

SSH 鍵の生成・追加・パスフレーズ変更を GUI から。AWS CodeCommit のサインインプロセスもネイティブに統合済みです。 Create, import and re-passphrase SSH keys from the GUI, with native AWS CodeCommit sign-in built in.

GitFlow · LFS · Submodules

GitFlow / LFS / Worktree / Submodule

GitFlow フロー、Git LFS の大容量ファイル、Worktree とサブモジュールの操作をワンクリックで。チーム運用の全部入りです。 GitFlow flows, Git LFS large files, worktrees and submodules — all the team-workflow staples, one click away.

Themes & Custom Fonts

5 種テーマ + JSON 上書き

Dark / Light に加え、OneDark / White を含む 5 種類のテーマ。JSON で色も等幅フォントも自分好みに上書きでき、Velopack による自動更新も標準搭載です。 Five built-in themes including OneDark and White, plus full JSON overrides for colors and monospace fonts. Velopack auto-update comes standard.

How it works

いつもの Git を、もっと静かに。

Your everyday Git, made quieter.

リポジトリを開くOpen a Git repository — local, SSH or HTTPS
↓   グラフ / Graph   ↓
ブランチ・差分・履歴Branch, diff, history — all in one view
↓   コミット / Commit   ↓
AI が下書き、人が仕上げるAI drafts the message, you finalize and push
  1. 1
    リポジトリを開く Open a repository.
    ローカル、SSH、HTTPS、AWS CodeCommit — どれでも同じ操作感で。
  2. 2
    グラフから操作する Drive from the graph.
    ブランチ、マージ、リベース、cherry-pick まで右クリックひとつ。
  3. 3
    AI が文面、人が承認 Commit with AI, ship in calm.
    差分から AI が下書きしたメッセージを確認してプッシュ。GPG / SSH / X.509 で署名も。
