機能 Features

日々の Git を、やわらかな光のなかで。

Everything you need for daily Git work — in warm, dappled light.

ビジュアルコミットグラフVisual Commit Graph

ベジェ曲線で描かれる 10 色のコミットグラフ。履歴・ブランチ・タグを一望できます。 A 10-colour commit graph drawn with smooth Bézier curves — history, branches and tags at a glance.

差分・Blame・画像差分Diff, Blame & Image Diff

サイドバイサイド／インライン差分、シンタックスハイライト、Blame、画像のオーバーレイ比較に対応。 Side-by-side and inline diffs, syntax highlighting, blame, plus overlay and swipe image comparison.

AI コミットメッセージAI Commit Messages

OpenAI・Azure OpenAI・Gemini・Anthropic を使い、差分から自動でメッセージを生成。 Generate commit messages from your diff with OpenAI, Azure OpenAI, Gemini or Anthropic.

SSH 鍵管理 & CodeCommitSSH Keys & CodeCommit

リモートごとの SSH 鍵選択と 3 段フォールバック。AWS CodeCommit（HTTPS / SSH / GRC）にも対応。 Per-remote SSH keys with 3-tier fallback, plus AWS CodeCommit (HTTPS / SSH / GRC) support.

17 言語 & 5 テーマ17 Languages & 5 Themes

翻訳率 100% の 17 言語と、Default / Light / Dark / White / OneDark の 5 テーマ。 17 fully-translated languages and five themes: Default, Light, Dark, White and OneDark.

貢献チャート & 高速起動Contribution Charts & Native AOT

著者ごとの貢献統計グラフを表示。Native AOT ビルドで軽快に起動します。 Per-author contribution charts, with Native AOT builds for fast, lightweight startup.
